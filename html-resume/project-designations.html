Description

This project involved developing an internal web application to automate the process of assigning and managing course designations within Canvas. Previously, these requests required significant manual review and coordination.

The tool streamlined submissions, validation, and approval workflows, allowing staff to process requests more consistently while reducing turnaround time for faculty.

  • Built using Flask with a SQL-backed database.
  • Automated validation and tracking of designation requests.
  • Designed for internal users with clarity and efficiency as priorities.

Skills Learned

  • Designing internal automation tools for real users
  • Backend development with Flask and relational databases
  • Translating business processes into technical workflows
  • Maintaining and iterating on production-adjacent systems
